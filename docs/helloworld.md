# !helloworld.md

> Note: this file is always written by Jatai into a newly created node INBOX during first-time auto-onboarding.

Welcome to Jatai.

Your node is ready to use. Jatai is file-system first: drop files in OUTBOX to broadcast, and read files from INBOX.

## Quick next steps

1. Check your node status:

```bash
jatai status
```

2. Start the daemon:

```bash
jatai start
```

3. Read docs in terminal:

```bash
jatai docs
jatai docs quickstart
```

4. Export docs to your INBOX when needed:

```bash
jatai docs -i
jatai docs quickstart -i
```
