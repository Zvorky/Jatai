# Quickstart

Jatai is a local micro-email bus based on folders.

## Basic flow

1. Run `jatai init` in a project folder.
2. Put markdown files in `OUTBOX`.
3. Read incoming files in `INBOX`.
4. Start the daemon with `jatai start`.

## Useful commands

- `jatai status`
- `jatai docs`
- `jatai docs retry`
